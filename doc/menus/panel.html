Panel Menu

Edit...

 (This is a sub-type of Skurba Panel, allowing things to be changed, but not re-ordered. It can be distinguished by the presence of the work "(Edit)" in the Key Box (in the upper right corner of the panel).
Items Allows changing the names assigned to individual items and their interactions with other items.

Note that invoking this panel will cause the list of maximal cliques to be recreated and their order and names to be reset to program defaults.

Cliques Allows changing the names assigned to groups of interacting items

Shkurba...

 (This is a sub-type of Skurba Panel, allowing things to be re-ordered, but not changed. It can be distinguished by the presence of the work "(Shkurba)" in the Key Box (in the upper right corner of the panel).
Items Displays a matrix showing the interactions between the items and allows the user to rearrange order of items to produce "arrowhead" patterns described in the Shkurba method of problem solution. To move an item (either vertically or horizontally), just drag it to the new location.
Cliques Displays a matrix showing the maximal cliques (rows) and the items (columns) which belong to them. Allows the user to rearrange order of cliques (groups of interacting items) or the order of the items. This can be used in the traditional solution method of recursively ordering the cliques.

Display...

 Textual display of the problem components (items or cliques)
Items Displays a list of the items and the items which each interacts with.
Cliques Displays a list of the cliques and the items which are members of each.

Graph...

 Drawing of graph where the items are nodes.
Intersection Displays graph where the edges are drawn between interacting items (nodes).
Complementary Displays graph where the edges are drawn between non-interacting items (nodes).
