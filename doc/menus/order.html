Order Menu

Sort Order...

 Sort items or cliques in a manner which often moves the order of the items closer to that of a solution.
Sort Rows Up Move the item with the fewest interacting items (or the clique with the fewest members) to the top of the display and work downward.
Sort Rows Down Move the item with the fewest interacting items (or the clique with the fewest members) to the bottom of the display and work upward.
Sort Columns Right Move the item with the fewest interacting items (or belonging to the fewest cliques) to the right of the display and work toward the left.
Sort Columns Left Move the item with the fewest interacting items (or belonging to the fewest cliques) to the left of the display and work toward the right.

Reverse Order...
Reverse Rows Reverse the order of the rows.
Reverse Columns Reverse the order of the columns.
Reverse Both Reverse the order of both the rows and the columns.

Undo Move

 Selecting this (or typing <cntl-z>) will undo the last move made on the current Shkurba panel. You can continue undoing previous moves. Sort, Reverse or Reset Order are not affected.
The Move history will be cleared (and you will be unable to undo previous moves), if you
  1. Change panels (or select the same one again)
  2. Reverse or Sort the rows or columns
  3. Reset the row or column order

Reset Order...
Reset Row Order Reset the order of the rows to what it was when the problem was first created.
Reset Column Order Reset the order of the columns to what it was when the problem was first created.
