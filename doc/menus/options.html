Options Menu

Note that when you leave the program, the current values of the options will be saved in a file named "javaBenzer.properties", which is saved in a foilder named "config" located in the folder where the "javaBenzer.jar" file is located. These values will be reloaded the next time the program is started. You can always reset the options to the program's default values by either removing individual lines from the file or by simply deleting the entire file.
Grayscale Checkbox allowing you to cause the program to display panels in shades of gray rather than color.
Select Colors... If you don't like the current color scheme, you can change it...
Select Matrix Colors
Select Text Color
Select Background Color
Cell Size This will display a slider allowing you to choose the width (and height) of each cell that will be displayed. The units are pixels, and allow you to customize the size of a display on your screen or on a printed output.
Text Justification... Allows you to choose the justification of the text labels for the rows or columns of the matrix.
Left Away from the left side of the matrix, or toward the bottom of the matrix.
Right Toward the left side of the matrix, or away from the bottom of the matrix.
Center Centered in the margin.
Note that the current setting will be selected.
Reset Options Change the current values of all the options back to what was read from the file when the program began running.
